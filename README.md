# Power BI Sales Dashboard

## Project Overview

This project is an end-to-end Power BI dashboard designed to analyze sales performance, customer behavior, and product trends. It helps business users make data-driven decisions using interactive visualizations.

---

## Objectives

* Track total sales, profit, and key performance metrics
* Analyze monthly and yearly sales trends
* Identify top customers and best-selling products
* Understand customer demographics and regional performance

---

## Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Excel (Data Source)

---

## Dataset Description

The dataset includes:

* **Sales Transactions** – Order details, sales amount, profit, quantity
* **Product Details** – Product category, price, stock availability
* **Customer Information** – Customer demographics and purchase history

---

## Dashboard Features

* Sales Overview (KPIs: Total Sales, Profit, Margin)
* Monthly Trends Analysis
* Top Customers & Products
* Profit & Revenue Analysis
* Dynamic Filters (Date, Region, Category)

---

## How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Refresh data if needed
4. Explore using filters and visuals

---

## Key Insights

* Identified top-performing products and customers
* Observed seasonal sales trends
* Analyzed profit margins across categories
